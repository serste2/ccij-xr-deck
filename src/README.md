# CCIJ XR Deck

A 6-slide React presentation on:

**“XR & Immersive Storytelling at CCIJ: from spatial reporting to playable systems”**

Built with **Vite + React**.

---

## Features

- 16:9 slide canvas
- One slide visible at a time
- Keyboard navigation (`←` / `→`)
- Subtle transitions
- 3 image placeholders per slide:
  - `slide1_img1`, `slide1_img2`, `slide1_img3`
  - ...
  - `slide6_img1`, `slide6_img2`, `slide6_img3`
- Caption field under each image slot
- Hidden expandable speaker notes per slide
- Editorial visual style (dark teal / warm yellow / deep red accents)
